\subsection{Detailed Description}
Implementation of \mbox{\hyperlink{interfacecom_1_1mysql_1_1cj_1_1protocol_1_1_value_decoder}{com.\+mysql.\+cj.\+protocol.\+Value\+Decoder}} for the My\+S\+QL text protocol. All values will be received as {\itshape Length\+Encoded\+String} values. 

Refer to My\+S\+QL documentation for format of values as strings. 

Numeric values are returned as A\+S\+C\+II (encoding=63/binary).
