Detailed Description

Implementation of com.mysql.cj.protocol.ValueDecoder for the MySQL text protocol. All values will be received as LengthEncodedString values.

Refer to MySQL documentation for format of values as strings.

Numeric values are returned as ASCII (encoding=63/binary).
