◆ configureClientCharacterSet()

boolean com.mysql.cj.NativeSession.configureClientCharacterSet ( boolean  dontCheckServerMatch)

Sets up client character set. This must be done before any further communication with the server!

Returns
true if this routine actually configured the client character set, or false if the driver needs to use 'older' methods to detect the character set, as it is connected to a MySQL server older than 4.1.0
Exceptions
CJExceptionif an exception happens while sending 'SET NAMES' to the server, or the server sends character set information that the client doesn't know about.

Check if we need a CharsetEncoder for escaping codepoints that are transformed to backslash (0x5c) in the connection encoding.
