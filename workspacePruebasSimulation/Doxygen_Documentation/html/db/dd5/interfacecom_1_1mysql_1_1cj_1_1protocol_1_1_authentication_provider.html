static byte com.mysql.cj.protocol.AuthenticationProvider< M extends Message >.getCharsetForHandshake ( String  enc,
ServerVersion  sv 
)
static

Get the MySQL collation index for the handshake packet. A single byte will be added to the packet corresponding to the collation index found for the requested Java encoding name.

If the index is > 255 which may be valid at some point in the future, an exception will be thrown. At the time of this implementation the index cannot be > 255 and only the COM_CHANGE_USER rpc, not the handshake response, can handle a value > 255.
