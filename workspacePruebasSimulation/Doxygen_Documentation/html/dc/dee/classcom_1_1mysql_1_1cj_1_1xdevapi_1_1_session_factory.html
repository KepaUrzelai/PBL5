Detailed Description

SessionFactory is used for creation of sessions.

SessionFactory xFactory = new SessionFactory();
Session session1 = xFactory.getSession("<b>mysqlx:</b>//[user1[:pwd1]@]host1[:port1]/db");
Session session2 = xFactory.getSession("<b>mysqlx:</b>//host2[:port2]/db?user=user2&amp;password=pwd2");
