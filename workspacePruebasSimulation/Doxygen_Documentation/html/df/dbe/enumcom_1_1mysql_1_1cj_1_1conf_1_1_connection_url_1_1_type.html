Detailed Description

The database URL type which is determined by the protocol section of the connection string.

◆ fromValue()

static Type com.mysql.cj.conf.ConnectionUrl.Type.fromValue ( String  protocol,
int  n 
)
static

Returns the Type corresponding to the given protocol and number of hosts, if any. Otherwise throws an IllegalArgumentException. Calling this method with the argument n lower than 0 skips the hosts cardinality validation. This should be used for URL protocol validation only as the returned Type won't won't reliable represent the database URL type.

Parameters
protocolthe protocol
nthe number of hosts in the database URL
