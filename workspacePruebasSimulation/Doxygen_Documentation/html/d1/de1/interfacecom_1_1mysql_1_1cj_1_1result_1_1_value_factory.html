Detailed Description

A class implements the ValueFactory<T> interface to create value instances from intermediate forms.

A ValueFactory implementation is responsible for creating instance of a single type, supplying a value for the type parameter T. If an instance cannot be created from the intermediate form, an java.lang.UnsupportedOperationException can be thrown.
